# Dev

1. Clonar el proyecto
1. Copiar el ```env.template``` y renombrar a ```.env```
2. Ejecutar
```
npm install
```
4. Levantar la image (Docker desktop)
```
docker-compose up -d
```
5. Levantar el backend de Nest
```
npm run start:dev
6. Visitar el sitio
```
localhost:3000/graphql
```
